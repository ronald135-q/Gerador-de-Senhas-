# Gerador-de-Senhas-
🛠Ferramentas e Tecnologias
HTML
CSS
Javascript

⚙️Funcionalidades
A aplicação permite que se obtenha alguma senha aleatória após clicar no botão "Gerar senha" para isso. Uma senha nova é gerada a partir de cada novo click no botão.
